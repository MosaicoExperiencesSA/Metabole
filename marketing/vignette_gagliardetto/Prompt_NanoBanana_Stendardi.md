# Stendardi MetaboleAI — Prompt per Nano Banana / Gemini

Uno stendardo (gagliardetto) per ogni membro della ciurma, nella veste che ti è piaciuta per Simone.

**Come usarlo:** incolla **un prompt alla volta** in Nano Banana (Gemini), imposta il formato **verticale 4:5** (ritratto), genera e, se serve, rigenera 2–3 volte scegliendo la resa migliore.

**Nota sul testo:** i modelli immagine a volte sbagliano le scritte lunghe in italiano. Le parole corte (nome, ruolo, "MetaboleAI", il motto "La nave è nuova. Ora si salpa.") vengono di solito bene; se la **frase lunga** esce storpiata, togli quella riga dal prompt e te la aggiungo io dopo con la scritta pulita.
